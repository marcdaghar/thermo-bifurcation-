# Thermodynamic Bifurcation in Debt-Based Monetary Systems

This repository contains the code for the paper "Thermodynamic Bifurcation in Debt-Based Monetary Systems: Parameter Λ and the Inevitability of Economic Erosion" by Marc Daghar.

## Overview

The model introduces a thermodynamic bifurcation parameter Λ that measures the ratio of debt service obligations to the rate of low-entropy extraction. The code demonstrates that under standard debt dynamics, Λ inevitably exceeds unity, leading to asymptotic economic erosion.
